Gather the personal details of the customer

- First Name (required)

- Last Name (required)

- Address (required)

- SSN / Drivers License (required)

- Issuance of State of SSN / Drivers License (required)

- State of residence

- Date of Birth

- Phone number

          If a Consumer does not have an SSN or DL issued in the US, but does have an Individual Tax Identification Number (ITIN), this can be used to attempt authentication. The ITIN is a nine-digit number, like an SSN, and should be entered into the SSN field. The ITIN should be used as a last resort and in combination with the other authentication requirements, such as full name and address.

          If a Consumer does not have SSN/DLN/ITIN, a Tribal Identification Card is an acceptable Government Issued ID. (Do not enter this EDN# in the SSN or DLN fields in MBS. Add in the notes).
